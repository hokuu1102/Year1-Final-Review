## 学校大学物理标准物理量符号汇总

### 力学与刚体转动
| 物理量   | 学校标准符号 | 备注（易混淆点） |
| -------- | ------------ | ---------------- |
| 角加速度 | **β**        | ✅ 禁用 α         |
| 转动惯量 | **J**        | ✅ 禁用 I         |
| 功       | **A**        | ✅ 禁用 W         |
| 角速度   | ω            | 无争议           |
| 力矩     | M            | 无争议           |
| 线加速度 | a            | 无争议           |
| 速度     | v            | 无争议           |
| 力       | F, f         | 无争议           |

### 热学
| 物理量 | 学校标准符号 | 备注     |
| ------ | ------------ | -------- |
| 热量   | Q            | 无争议   |
| 内能   | ΔE, ΔU       | 两者通用 |
| 熵     | S            | 无争议   |
| 压强   | p            | 无争议   |
| 温度   | T            | 无争议   |

### 波动与光学
| 物理量 | 学校标准符号 | 备注     |
| ------ | ------------ | -------- |
| 频率   | ν, f         | 两者通用 |
| 波长   | λ            | 无争议   |
| 波速   | u            | 无争议   |
| 相位   | φ            | 无争议   |
| 振幅   | A            | 无争议   |

---

**核心易错三点总结**：β（角加速度）、J（转动惯量）、A（功）
