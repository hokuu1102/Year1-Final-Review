## 三、各题型高频考点

### （一）单项选择题（15-20分）

基础概念辨析，陷阱多，侧重易混淆知识点：

| 考点 | 出现频率 | 典型问法 |
|------|:-------:|---------|
| 矩阵运算与可逆性 | 非常高 | "设 $A,B$ 都是 $n$ 阶非零矩阵，且 $AB=O$，则..." |
| 行列式性质 | 高 | "已知 $|A|=2$，则 $|2A|=$..." |
| 线性相关性 | 高 | "设向量组 $\alpha_1,\alpha_2,\alpha_3$ 线性相关..." |
| 相似矩阵 | 高 | "若 $A$ 与 $B$ 相似，则..." |
| 二次型基础 | 中 | "设二次型 $f=x^TAx$ 的矩阵为 $A$..." |
| 特征值性质 | 中 | "设 $A$ 的特征值为 $1,2$，则 $|A|=$..." |

### （二）填空题（20分）

计算类为主，侧重公式直接应用和快速计算：

| 考点 | 出现频率 | 典型问法 |
|------|:-------:|---------|
| 伴随矩阵与逆矩阵 | 非常高 | "已知 $A=...$，则 $A^*=$___" |
| 行列式计算 | 非常高 | "已知 $|A|=...$，则 $|A^{-1}-2A^*|=$___" |
| 特征值性质 | 高 | "矩阵 $A$ 的特征值为 $1,-1,2$，则 $|E-A^2|=$___" |
| 向量空间 | 高 | "由基(I)到基(II)的过渡矩阵 $P=$___" |
| 二次型参数 | 中 | "二次型 $f=...$ 的正惯性指数为___" |

### （三）解答题（60-65分）

分为**基础计算题**、**综合应用题**和**证明题**三类：

| 题型 | 分值 | 出现频率 | 核心考点 |
|------|:---:|:-------:|---------|
| 行列式计算 | 6-10分 | 每年 | 特殊行列式（循环、三对角、范德蒙）、按行/列展开、递推法 |
| 矩阵方程求解 | 8-10分 | 每年 | 伴随矩阵+逆矩阵综合运算，$ABA^*=...$ 型 |
| 含参线性方程组 | 10-15分 | 每年 | 增广矩阵行变换+参数讨论唯一解/无解/无穷多解 |
| 向量组线性相关性 | 8-10分 | 高频 | 判定线性相关/无关、求极大无关组、线性表示 |
| 特征值与对角化 | 12-15分 | 每年 | 求特征值特征向量、证明可对角化、相似应用 |
| 二次型 | 12-15分 | 每年 | 求参数→正交变换化标准形 |
| 向量空间 | 6-14分 | 高频 | 基变换公式、过渡矩阵、坐标计算 |
| 证明题 | 6-10分 | 每年 | 秩的不等式、正交矩阵、可对角化证明 |
| 应用题 | 6-8分 | 中频 | 投入产出模型、热传导方程组建模 |

## 四、三大核心模块命题规律

### 1. 特征值与相似对角化（15-25分/卷）

**必考内容**：
- 特征值性质：若 $\lambda$ 是 $A$ 的特征值，则 $f(\lambda)$ 是 $f(A)$ 的特征值
- $|A|=\lambda_1\lambda_2\cdots\lambda_n$，$\text{tr}(A)=\lambda_1+\lambda_2+\cdots+\lambda_n$
- 矩阵可对角化证明（高频证明题）
- 相似矩阵性质（选择题高频易错点）

**易错点**：
- 相似矩阵特征值相同，但特征向量不同
- $A$ 可对角化 $\Leftrightarrow$ $A$ 有 $n$ 个线性无关的特征向量

### 2. 二次型（12-15分/卷）

**固定流程**：
1. 根据二次型的秩/特征值/正交变换条件求参数
2. 用正交变换化二次型为标准形（施密特正交化+单位化）

**延伸考点**：
- 正惯性指数、负惯性指数
- 正定矩阵判定
- 二次型与二次曲面的对应

### 3. 向量空间（6-14分/卷）

**核心公式**：
- 基变换：$(b_1,b_2,b_3)=(a_1,a_2,a_3)P$（过渡矩阵 $P$）
- 坐标变换：$y=P^{-1}x$

**题型固定**：已知坐标变换关系求过渡矩阵，或已知向量在一个基下的坐标求另一个基下的坐标。

## 五、整体出题风格

1. **题型稳定，考点固定**：近三年题型、题量、分值分布几乎无变化，核心考点重复率80%+
2. **基础与综合并重**：既有概念辨析基础题，也有结合多知识点的综合题
3. **计算量较大**：行列式、矩阵运算、正交变换对计算准确性要求高
4. **证明题侧重核心定理**：围绕矩阵秩、可对角化、正定矩阵，不考偏题怪题
5. **特色应用导向**：加入线性代数在工程中的应用题目（投入产出、热传导）
6. **易错点针对性强**：频繁考察易混淆知识点（代数余子式vs余子式、相似vs合同）

## 六、编写题目时的注意事项

### 选择题编写规范
- 考概念辨析，不考硬算
- 每题4个选项，干扰项要有迷惑性
- 重点考察：矩阵运算律、伴随矩阵性质、线性相关性判断

### 填空题编写规范
- 需要动手计算，结果简洁（整数或简单分数）
- 考察公式直接应用和快速计算能力

### 解答题编写规范
- 步骤要完整，过程分很重要
- 每道题有明确的得分点
- 含参讨论题要列出所有情况

### 证明题编写规范
- 围绕核心定理（秩的不等式、可对角化、正定矩阵）
- 不考偏题怪题
- 区分平行班和实验班难度
