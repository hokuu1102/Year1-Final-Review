## 一、行列式

### 1.1 基本计算

| 阶数 | 公式 |
|:---:|------|
| 二阶 | $\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc$ |
| 三阶 | 沙路法：↘ 三项取正，↙ 三项取负 |
| n 阶 | $\|A\| = \sum (-1)^{\tau} a_{1p_1} a_{2p_2} \cdots a_{np_n}$（逆序数定义） |

**代数余子式**：$A_{ij} = (-1)^{i+j} M_{ij}$

**展开定理**：
- 按行：$\|A\| = \sum_{j=1}^{n} a_{ij} A_{ij}$
- 按列：$\|A\| = \sum_{i=1}^{n} a_{ij} A_{ij}$

### 1.2 性质速查

| 性质 | 公式 |
|------|------|
| 转置 | $\|A^T\| = \|A\|$ |
| 数乘 | $\|kA\| = k^{\color{red}n}\|A\|$（$n$ 为阶数） |
| 乘积 | $\|AB\| = \|A\| \cdot \|B\|$ |
| 逆 | $\|A^{-1}\| = \dfrac{1}{\|A\|}$ |
| 伴随 | $\|A^*\| = \|A\|^{n-1}$ |
| 幂 | $\|A^k\| = \|A\|^k$ |
| 换行 | 两行交换，行列式**变号** |
| 倍加 | 某行 + 另一行的 $k$ 倍，值**不变** |
| 成比例 | 两行相同/成比例 → $\|A\| = 0$ |

### 1.3 特殊行列式公式

| 名称 | 公式 | 解法关键 |
|------|------|---------|
| 一杠一星 | 对角乘积 $\times$ 非零元系数 | 按额外非零元所在行/列展开 |
| 两杠一星 | $[1 + (-1)^{n+1}] n!$（例题结论） | 按第一行展开 → 两个三角行列式 |
| 箭头型（爪型） | $D_n = \left(\prod_{i=2}^n a_i\right)\left(a_1 - \sum_{i=2}^n \frac{b_i c_i}{a_i}\right)$ | 打洞消元：对角线消去第一列 |
| 类箭头型 | 同箭头型 | 打洞消元 / 加边化为箭头型 |
| ab 型 | $D_n = [b+(n-1)a](b-a)^{n-1}$ | 列全加到第 1 列 → 提公因子 → 上三角 |
| 范德蒙 | $V_n = \prod_{1 \leqslant j < i \leqslant n} (x_i - x_j)$ | 后减前全乘；三阶 $(x_3-x_2)(x_3-x_1)(x_2-x_1)$ |
| 三对角（两杠一星·递推型） | $D_n = aD_{n-1} - bcD_{n-2}$ | 按第一行展开 → 特征方程 $r^2 - ar + bc = 0$ |
| 循环三阶 | $a^3 + b^3 + c^3 - 3abc$ | 沙路法或直接套公式 |

### 1.4 克莱姆法则

- $Ax = b$ 有唯一解 $\Leftrightarrow$ $\|A\| \neq 0$
- $x_j = \dfrac{D_j}{D}$，$D_j$ 是将 $A$ 第 $j$ 列换为 $b$

**齐次 $Ax = 0$**：
- $\|A\| \neq 0$ → 只有零解
- $\|A\| = 0$ → 有非零解

> ⚠️ 仅适用于方程数 = 未知数个数（方阵）

**代数余子式求和**：$\sum k_j A_{ij}$ = 将第 $i$ 行换成 $(k_1, k_2, \ldots)$ 后再算行列式

## 二、矩阵

### 2.1 特殊矩阵

| 名称 | 定义 |
|------|------|
| 零矩阵 $O$ | 全 0 |
| 单位矩阵 $E$ | 对角全 1，其余 0 |
| 对角矩阵 | 非对角全 0 |
| 对称矩阵 | $A^T = A$ |
| 反对称矩阵 | $A^T = -A$，对角必为 0 |

### 2.2 运算律

| 运算 | 交换律 | 结合律 | 分配律 |
|------|:---:|:---:|:---:|
| 加法 | ✅ | ✅ | ✅ |
| 乘法 | ❌ | ✅ | 单侧 |
| 数乘 | ✅ | ✅ | ✅ |

> ⚠️ $(A+B)^2 = A^2 + AB + BA + B^2$（不是 $A^2 + 2AB + B^2$）

### 2.3 上标运算规律

**穿过乘法，顺序反转**：
$$(AB)^T = B^T A^T, \qquad (AB)^{-1} = B^{-1}A^{-1}$$

**上标之间可交换**：
$$(A^T)^{-1} = (A^{-1})^T, \qquad (A^T)^k = (A^k)^T, \qquad (A^{-1})^k = (A^k)^{-1}$$

| 组合 | 公式 |
|------|------|
| $(A^T)^{-1}$ | $= (A^{-1})^T$ |
| $(A^T)^n$ | $= (A^n)^T$ |
| $(A^{-1})^k$ | $= (A^k)^{-1}$ |
| $(A^*)^T$ | $= (A^T)^*$ |

### 2.4 方阵行列式

$$|A^T| = |A|, \quad |kA| = k^n|A|, \quad |AB| = |A||B|, \quad |A^k| = |A|^k$$

### 2.5 逆矩阵

**可逆条件**：$\|A\| \neq 0$

**求法一（伴随矩阵法）**：$A^{-1} = \dfrac{1}{\|A\|} A^*$

**求法二（初等行变换）**：$(A \mid E) \xrightarrow{\text{行变换}} (E \mid A^{-1})$

**二阶速算**（口诀：主对角互换，副对角变号，除以行列式）：
$$\begin{pmatrix} a & b \\ c & d \end{pmatrix}^{-1} = \frac{1}{ad-bc}\begin{pmatrix} d & -b \\ -c & a \end{pmatrix}$$

**常用逆**：
| 矩阵 | 逆 |
|------|-----|
| $kE$ | $\frac{1}{k}E$ |
| $\text{diag}(d_1,\ldots,d_n)$ | $\text{diag}(\frac{1}{d_1},\ldots,\frac{1}{d_n})$ |
| 分块对角 $\begin{pmatrix} A & O \\ O & B \end{pmatrix}$ | $\begin{pmatrix} A^{-1} & O \\ O & B^{-1} \end{pmatrix}$ |

**抽象矩阵证可逆**：从 $A^2 + aA + bE = O$ 中凑出 $A \cdot (\cdots) = kE$（$k \neq 0$）

### 2.6 伴随矩阵

| 性质 | 公式 |
|------|------|
| 基本恒等式 | $AA^* = A^*A = \|A\|E$ |
| 行列式 | $\|A^*\| = \|A\|^{n-1}$ |
| 逆 | $(A^*)^{-1} = \dfrac{A}{\|A\|}$ |
| 数乘 | $(kA)^* = k^{\color{red}n-1} A^*$ |
| 伴随的伴随 | $(A^*)^* = \|A\|^{n-2} A$（$n \geqslant 2$） |
| 乘积 | $(AB)^* = B^*A^*$ |

> ⚠️ $A^*_{ij} = A_{ji}$（放的是 $A_{ji}$，不是 $A_{ij}$）——构造时必须**转置**！

### 2.7 分块矩阵

| 结构 | 行列式 | 逆 |
|------|--------|-----|
| $\begin{pmatrix} A & O \\ O & B \end{pmatrix}$ | $\|A\| \cdot \|B\|$ | $\begin{pmatrix} A^{-1} & O \\ O & B^{-1} \end{pmatrix}$ |
| $\begin{pmatrix} A & C \\ O & B \end{pmatrix}$ | $\|A\| \cdot \|B\|$ | $\begin{pmatrix} A^{-1} & -A^{-1}CB^{-1} \\ O & B^{-1} \end{pmatrix}$ |
| $\begin{pmatrix} A & O \\ C & B \end{pmatrix}$ | $\|A\| \cdot \|B\|$ | $\begin{pmatrix} A^{-1} & O \\ -B^{-1}CA^{-1} & B^{-1} \end{pmatrix}$ |

**转置**：$\begin{pmatrix} A & B \\ C & D \end{pmatrix}^T = \begin{pmatrix} A^T & C^T \\ B^T & D^T \end{pmatrix}$

### 2.8 初等矩阵

| 变换 | 初等矩阵 | 逆 |
|------|---------|-----|
| $r_i \leftrightarrow r_j$ | $E(i,j)$ | $E(i,j)$ |
| $r_i \times k$ | $E(i(k))$ | $E(i(1/k))$ |
| $r_i + k r_j$ | $E(ij(k))$ | $E(ij(-k))$ |

> **左乘行变，右乘列变**

### 2.9 矩阵方程

| 方程 | 解 |
|------|-----|
| $AX = B$ | $X = A^{-1}B$（$A$ 可逆） |
| $XA = B$ | $X = BA^{-1}$（$A$ 可逆） |
| $AXB = C$ | $X = A^{-1}CB^{-1}$（$A, B$ 可逆） |

> ⚠️ 乘法不交换：$AX = B$ 的解是 $A^{-1}B$，**不是** $BA^{-1}$

**$ABA^*$ 型方程套路**：方程含 $A^*$ → 两边同乘 $A$（利用 $A^*A = \|A\|E$）→ 化 $A^*$ 为 $\|A\|A^{-1}$ → 解常规矩阵方程

## 三、矩阵的秩与向量组

### 3.1 秩的定义

$r(A)$ = 最高阶非零子式的阶数 = 行阶梯形中非零行的行数

### 3.2 秩的性质

| 性质 | 公式 |
|------|------|
| 零矩阵 | $r(O) = 0$ |
| 满秩（可逆） | $r(A) = n$ |
| 转置 | $r(A^T) = r(A)$ |
| 乘积上界 | $r(AB) \leqslant \min(r(A), r(B))$ |
| 乘积下界 | $r(A) + r(B) - n \leqslant r(AB)$ |
| 和的界 | $r(A+B) \leqslant r(A) + r(B)$ |
| $AB = O$ 时 | $r(A) + r(B) \leqslant n$ |
| $AB = E_m$ 时 | $r(A) = m$，$r(B) = m$ |
| 行满秩 $r(A) = m$ | $Ax = b$ 一定有解 |
| $R(E - AB) + R(E + AB)$ | $\geqslant n$（西电高频） |

### 3.3 线性相关性判定

| 方法 | 条件 | 结论 |
|------|------|------|
| 方阵行列式 | $\|A\| = 0$ | 线性相关 |
| 方阵行列式 | $\|A\| \neq 0$ | 线性无关 |
| 秩法 | $r(A) <$ 列数 | 线性相关 |
| 秩法 | $r(A) =$ 列数 | 线性无关 |

> ⚠️ 向量个数 > 维数 → 必线性相关
>
> ⚠️ 大组无关 → 小组无关；小组无关 ⇏ 大组无关

### 3.4 抽象向量组拆分

$$(\vec{\beta}_1, \ldots, \vec{\beta}_s) = (\vec{\alpha}_1, \ldots, \vec{\alpha}_t) C$$

| $C$ 列满秩 $r(C) = s$ | $\vec{\beta}$ 线性**无关** |
| $C$ 列不满秩 $r(C) < s$ | $\vec{\beta}$ 线性**相关** |

> 无关组 × 可逆阵 → 无关；无关组 × 不可逆阵 → 有关

### 3.5 极大无关组求法

1. 以向量为列构造矩阵 $A$
2. 行变换 → 行阶梯形
3. 拐弯处（主元所在列）对应的**原 $A$ 的列** → 极大无关组

## 四、线性方程组

### 4.1 齐次方程组 $Ax = 0$

| 条件 | 解的情况 |
|------|---------|
| $r(A) = n$ | 只有零解 |
| $r(A) < n$ | 有非零解（无穷多解） |

**基础解系向量个数**：$n - r(A)$（= 自由变量个数）

**通解**：$x = k_1\vec{\xi}_1 + k_2\vec{\xi}_2 + \cdots + k_{n-r}\vec{\xi}_{n-r}$

### 4.2 非齐次方程组 $Ax = b$

| 条件 | 解的情况 |
|------|---------|
| $r(A) = r(\bar{A}) = n$ | **唯一解** |
| $r(A) = r(\bar{A}) < n$ | **无穷多解** |
| $r(A) \neq r(\bar{A})$ | **无解** |

> $\bar{A} = (A \mid b)$ 是增广矩阵

**通解结构**：
$$\text{非齐次通解} = \text{齐次通解} + \text{非齐次特解}$$

**求特解**：自由变量全部取 0

### 4.3 解向量组合规律

若 $\eta_1, \eta_2, \eta_3$ 是 $Ax = b$ 的解：

| 组合系数和 | 结果 |
|-----------|------|
| $k_1 + k_2 + k_3 = 0$ | 是 $Ax = 0$ 的解 |
| $k_1 + k_2 + k_3 = 1$ | 是 $Ax = b$ 的解 |

## 五、特征值与特征向量

### 5.1 定义

$$A\vec{\alpha} = \lambda\vec{\alpha} \quad (\vec{\alpha} \neq \vec{0})$$

**求法**：
1. 特征方程：$\|\lambda E - A\| = 0$ → 得 $\lambda$
2. 解 $(\lambda E - A)x = 0$ → 得特征向量

### 5.2 特征值性质

| 性质 | 公式 |
|------|------|
| 行列式 | $\|A\| = \lambda_1 \lambda_2 \cdots \lambda_n$ |
| 迹 | $\text{tr}(A) = \lambda_1 + \lambda_2 + \cdots + \lambda_n = \sum a_{ii}$ |
| 多项式 $f(A)$ | 特征值为 $f(\lambda)$ |
| $A^{-1}$ 的特征值 | $1/\lambda$ |
| $A^*$ 的特征值 | $\|A\| / \lambda$ |
| $A^T$ 的特征值 | $\lambda$（相同） |
| 相似 $P^{-1}AP$ | 特征值仍为 $\lambda$，特征向量变为 $P^{-1}\vec{\alpha}$ |

### 5.3 重数

- **代数重数**：$\lambda$ 作为特征方程根的重数
- **几何重数**：$(\lambda E - A)x = 0$ 基础解系中向量个数
- **关系**：$1 \leqslant$ 几何重数 $\leqslant$ 代数重数

> ⚠️ $A$ 可对角化 $\Leftrightarrow$ 每个特征值的几何重数 = 代数重数

## 六、相似对角化

### 6.1 相似矩阵

$B = P^{-1}AP$，则 $A$ 与 $B$ 相似

| 不变量 | 说明 |
|--------|------|
| 特征值相同 | $\lambda(A) = \lambda(B)$ |
| 行列式相同 | $\|A\| = \|B\|$ |
| 迹相同 | $\text{tr}(A) = \text{tr}(B)$ |
| 秩相同 | $r(A) = r(B)$ |

### 6.2 可对角化条件

$A$ 可对角化 $\Leftrightarrow$ $A$ 有 $n$ 个线性无关的特征向量

**对角化步骤**：
1. 求特征值
2. 求特征向量
3. $P = (\vec{\alpha}_1, \ldots, \vec{\alpha}_n)$
4. $\Lambda = P^{-1}AP = \text{diag}(\lambda_1, \ldots, \lambda_n)$

### 6.3 实对称矩阵对角化

**性质**：
- 特征值全为实数
- 不同特征值的特征向量**正交**
- 一定可以**正交对角化**

**正交对角化步骤**：
1. 求特征值 → 特征向量
2. 同一特征值的特征向量 → **施密特正交化**
3. **单位化**所有特征向量
4. $Q = (\vec{e}_1, \ldots, \vec{e}_n)$（正交矩阵）
5. $Q^T A Q = \Lambda$

### 6.4 正交矩阵

$Q^T Q = QQ^T = E$，即 $Q^{-1} = Q^T$

- 列向量两两正交且为单位向量
- $\|Q\vec{x}\| = \|\vec{x}\|$（保范性）
- $\|Q\| = \pm 1$

### 6.5 施密特正交化

$$\vec{\beta}_1 = \vec{\alpha}_1$$

$$\vec{\beta}_2 = \vec{\alpha}_2 - \frac{(\vec{\alpha}_2, \vec{\beta}_1)}{(\vec{\beta}_1, \vec{\beta}_1)} \vec{\beta}_1$$

$$\vec{\beta}_3 = \vec{\alpha}_3 - \frac{(\vec{\alpha}_3, \vec{\beta}_1)}{(\vec{\beta}_1, \vec{\beta}_1)} \vec{\beta}_1 - \frac{(\vec{\alpha}_3, \vec{\beta}_2)}{(\vec{\beta}_2, \vec{\beta}_2)} \vec{\beta}_2$$

**单位化**：$\vec{e}_i = \dfrac{\vec{\beta}_i}{\|\vec{\beta}_i\|}$

> ⚠️ 只对**同一特征值**的特征向量使用施密特正交化（不同特征值的已自动正交）

### 6.6 可对角化证明（精简模板）

| 题型 | 关键思路 |
|------|---------|
| $A^2 = A$（幂等） | $A(A-E)=O$ → 特征值仅 $0,1$ → 两个特征空间直和 = 全空间 → 可对角化 |
| $A^2 = E$（对合） | $(A-E)(A+E)=O$ → 特征值仅 $\pm 1$ → 同理 |
| $n$ 个不同特征值 | 直接可对角化（充分非必要） |
| 一般 | 证每个特征值的几何重数 = 代数重数 |

### 6.7 正交矩阵证明（精简模板）

证 $Q^T Q = E$：列向量两两正交 + 各自长度为 1。

常用技巧：实对称不同特征值的特征向量自动正交，再单位化即得正交矩阵。

## 七、二次型

### 7.1 矩阵表示

$$f = \vec{x}^T A \vec{x}, \quad A \text{ 为对称矩阵}$$

**写 $A$ 的规则**：
- 主对角线 = 平方项系数
- $a_{ij} = a_{ji} = \dfrac{1}{2} \times$（$x_i x_j$ 的系数）

> 对任意实矩阵 $A$，$f = \vec{x}^T A \vec{x} = \vec{x}^T \cdot \dfrac{1}{2}(A + A^T) \cdot \vec{x}$

### 7.2 化标准形

**配方法**：逐个变量配方 → 换元 → $f = d_1y_1^2 + d_2y_2^2 + \cdots + d_ry_r^2$

**正交变换法**：$x = Qy$ → $f = \lambda_1y_1^2 + \cdots + \lambda_ny_n^2$（系数 = 特征值）

### 7.3 规范形

$$f = z_1^2 + \cdots + z_p^2 - z_{p+1}^2 - \cdots - z_{p+q}^2$$

| 概念 | 含义 |
|------|------|
| 正惯性指数 $p$ | 规范形中正项个数 |
| 负惯性指数 $q$ | 规范形中负项个数 |
| 符号差 | $p - q$ |
| 秩 | $r = p + q$ |

> ⚠️ 规范形的系数只能是 $+1$ 或 $-1$

### 7.4 正定判定

| 方法 | 条件 |
|------|------|
| 特征值法 | 所有特征值 $> 0$ |
| 顺序主子式法 | 各阶顺序主子式 $> 0$ |
| 惯性指数法 | 正惯性指数 $p = n$ |

> ⚠️ $\|A\| > 0$ ⇏ $A$ 正定（反例：$\begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix}$）

### 7.5 合同矩阵

$B = C^T A C$（$C$ 可逆），则 $A$ 与 $B$ 合同

**判定**：实对称矩阵合同 $\Leftrightarrow$ 正负惯性指数相同

| 关系 | 说明 |
|------|------|
| 相似 + 实对称 | ⇒ 合同 |
| 合同 | ⇏ 相似 |
| 相似看特征值 | 合同看惯性指数 |

### 7.6 二次型与二次曲面对应

| 标准形 | 曲面 | $(p,q)$ |
|------|------|:---:|
| $\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1$ | 椭球面 | $(3,0)$ |
| $\frac{x^2}{a^2}+\frac{y^2}{b^2}-\frac{z^2}{c^2}=1$ | 单叶双曲面 | $(2,1)$ |
| $\frac{x^2}{a^2}+\frac{y^2}{b^2}=2z$ | 椭圆抛物面 | $(2,0)$ |
| $\frac{x^2}{a^2}-\frac{y^2}{b^2}=2z$ | 马鞍面 | $(1,1)$ |

## 八、向量空间

### 8.1 基底与坐标

向量 = 基底 × 坐标：
$$\vec{v} = x_1\vec{\alpha}_1 + x_2\vec{\alpha}_2 + \cdots + x_n\vec{\alpha}_n$$

- 空间维数 = 基底中向量个数 = $r(A)$

### 8.2 过渡矩阵

$$(\beta_1, \ldots, \beta_n) = (\alpha_1, \ldots, \alpha_n) P$$

**求法**：$P = A^{-1}B$（$A$ 为 $\alpha$ 坐标矩阵，$B$ 为 $\beta$ 坐标矩阵）

### 8.3 坐标变换

$$y = P^{-1}x$$

- $x$：旧基 $\alpha$ 下的坐标
- $y$：新基 $\beta$ 下的坐标
- $P$：从 $\alpha$ 到 $\beta$ 的过渡矩阵

### 8.4 标准正交基下坐标

$$x_i = (\vec{v}, \vec{e}_i) \quad \text{（坐标 = 内积）}$$

### 8.5 正交基

| 概念 | 定义 |
|------|------|
| 正交基 | 向量两两正交 |
| 标准正交基 | 正交 + 每个向量是单位向量 |

## 九、高频易错对比表

| ❌ 错误 | ✅ 正确 |
|---------|---------|
| $\|kA\| = k\|A\|$ | $k^{\color{red}n}\|A\|$ |
| $(AB)^T = A^T B^T$ | $B^T A^T$ |
| $(AB)^{-1} = A^{-1}B^{-1}$ | $B^{-1}A^{-1}$ |
| $(A+B)^2 = A^2 + 2AB + B^2$ | $A^2 + AB + BA + B^2$ |
| $(kA)^* = kA^*$ | $k^{\color{red}n-1}A^*$ |
| $A^*_{ij} = A_{ij}$ | $A^*_{ij} = A_{\color{red}ji}$（必须转置！） |
| 分块矩阵逆 = 各块取逆 | 仅分块对角成立 |
| $\|A\| > 0$ ⇒ $A$ 正定 | 需要全部特征值 $> 0$ |
| 沙路法可用于四阶 | 仅三阶 |
| $AB = O$ ⇒ $A=O$ 或 $B=O$ | 不能推出（零因子存在） |
| $AB = AC$ ⇒ $B = C$ | 不能消去（$A$ 不一定可逆） |
| $R(A+B) = R(A) + R(B)$ | $\leqslant$，不是等号 |
| 范德蒙 $(x_i - x_j)$ | 应为 $\prod_{i < j} (x_{\color{red}j} - x_{\color{red}i})$（后面减前面） |
| 反对称奇数阶 $\|A\| \neq 0$ | $\|A\| = 0$（$n$ 为奇数时） |

## 十、终极口诀

| 模块 | 口诀 |
|------|------|
| 行列式 | $\|kA\|$ 乘 $k^n$；换行变号；成比例得零；沙路法仅三阶 |
| 矩阵 | 转置/逆穿过乘法 → 顺序反转；乘法不交换；$A^*$ 自带转置 |
| 伴随 | 余子式转置，自带 $\|A\|$ 缩放；数乘升 $n-1$ 次；$A^*A=\|A\|E$ |
| 秩 | $AB=O$ ⇒ 秩和 $\leqslant n$；乘积秩 $\leqslant$ 各因子秩；行满秩必有解 |
| 方程组 | 有解看增广秩相等；几解看秩等于 $n$；通解 = 齐通 + 非奇特 |
| 特征值 | 积 = 行列式，和 = 迹；$f(A)$ 特征值 = $f(\lambda)$；$A^*$ 特征值 = $\|A\|/\lambda$ |
| 相似 | 特征值相同，特征向量不同；换皮不换骨；实对称必可正交对角化 |
| 二次型 | 平方系数放对角，交叉系数一半放两边；规范形系数仅 $\pm 1$ |
| 合同 | 相似看特征值，合同看惯性指数；惯性指数相同即合同 |
| 证明 | 可对角化 → 凑 $n$ 个无关特征向量；正交 → 列正交 + 单位长 |
| 特殊行列式 | 循环三阶 $a^3+b^3+c^3-3abc$；范德蒙"后减前全乘"；ab型 $(a-b)^{n-1}$ |
