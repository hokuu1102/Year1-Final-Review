    行阶梯形矩阵的特点（辨析）：
        台阶可以长，但是不能一步跨两行
        下两行全为0也是阶梯型
    利用行初等变换 A -> 阶梯型B r(A) = B中非0行的行数

向量组的线性相关性：
    向量线性相关 - 对应成比例 - 向量组行列式 = 0
    向量组矩阵的秩小于列数->相关
     = ->无关
    抽象向量组的表达方式：
        复杂向量组 = 无关简单向量组 * C
        |C| = 0  <-> 相关
        即 无关组 * 可逆阵 -> 无关
        无关组 * 不可逆阵 -> 有关
    求向量组的秩和极大无关组：
        向量组的秩 = 矩阵的阶梯形中非零行数
        极大无关组一般取阶梯形中拐弯处所在的列向量

线性方程组：
    齐次方程组的求解：
        非零解（无穷解：自由变量任意取值） 和零解（唯一解）的充要条件
        基础解系：
            Ax = 0有无穷解时，解集的极大无关组为基础解系，所含解向量个数 = n -r(A) = 自由变量的个数
            求法：把系数阵化为行最简形（行最简形定义） -> 非拐弯处作为自由变量 -> 对应列取相反数，自由变量取值如1 0、0 1 ...
    非齐次方程组的求解：
        无穷解、唯一解、无解：系数阵与增广阵的秩，与列数n比较
        非齐通解= 齐通 + 非齐特
        齐通：基础解系
        非齐特：自由变量取0，其他取增广矩阵最后一列
    带参数方程组的求解：
        Ax = b有数个不同解->存在无穷解

二次型：
    二次型矩阵写法
    二次型的矩阵三要素：
        对称矩阵
        主对角元素为平方项系数
        其他元素为交叉项系数的一半（沿主对角线对称分布 ij <-> ji）  
    二次型的秩 = r(A)
    化二次型为标准型：
        配方法：
            构造完全平方
            先配所有含x1的项（把x2,x3..看作整体）->x2->x3...
            换元得标准形
        正交变换法：
            矩阵特征值、特征向量
            施密特正交化
            单位化得正交矩阵
            作变换令 x = Qy得标准形
    正定二次型和正定矩阵：
        二次型的判定方法：
            特征值全大于0（f的正惯性指数 = 未知数个数（化为标准形后正平方项的个数））
            各阶顺序主子式全大于0

向量空间：
    基底（不共线不共面，向量之间不能相互表示——不线性相关）
    坐标（系数）
    矩阵的秩表示了向量组的空间维数
    最大无关组（主元位置对应的列向量）
    线性相关 <-> 秩 != n <-> 行列式=0
    向量 = 基底 * 坐标   
    由向量求基底：
        写出坐标矩阵，移过去求逆矩阵
    基变换：
        两侧左乘新基底的逆矩阵
    正交基/施密特正交化：
        两两正交：正交基
        且若都为单位向量（内积为1）：标准正交基
        点乘即得系数
        施密特正交化： a1-a1在b1上的投影
